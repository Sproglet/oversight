OverSight Installer - simple low maintenance index for unpak.sh

Installation will build the initial index by scanning the following folders:
/share/

This requires internet access (to get program details) and may take 5-10 minutes initially.

When the link is selected the screen will pause for 1 minute, then you can return to the web services screen to view the index. however the index will not be ready for a few more minutes.
This will also update your unpak script
